HTTP Prompt
===========

|PyPI| |Docs| |Build| |Coverage| |Gitter|

HTTP Prompt is an interactive command-line HTTP client featuring autocomplete
and syntax highlighting, built on HTTPie_ and prompt_toolkit_.
